Release Highlights

What changed in 4.0

Information below is remastered from README and release notes so the website reflects the current stable release.

Transformation possession core

Possess and control mobs with full movement, camera adaptation, and attack timing systems.

Herobrine event systems

Scare-driven Herobrine sequences, shrine interactions, and horror encounter mechanics are now core identity features.

Hunter mode gameplay

Refined hunter behavior and combat pacing support intense chase-focused sessions.

Mini Prop Hunt minigame

Round flow, border enforcement, reward logic, and hint pacing were remastered for consistent match quality.

Speedrunner vs Hunter mode

Asymmetric chase experience with aggressive pursuit pressure and survival decision-making.

Companion bot AI for survival

Inbuilt companion bot is intelligent and brave, helping survival players with combat support and tactical follow behavior.

Fun and prank remaster

Timed morph pranks and command expansions like boing, confetti, and swap increase social chaos gameplay.

Transformation possession mechanisms

Aeonis is centered on direct possession play, where each entity behaves with tailored control logic.

  • First-person possession camera and interaction flow.
  • Mob-specific movement behavior for land, flight, and utility entities.
  • Special attack cooldown balancing for high-impact forms like Wither and Dragon.

Active minigame and encounter focus

Current stable spotlight is on replayable gameplay loops rather than WIP biome expansion.

  • Mini Prop Hunt with stronger state checks and round-end reliability.
  • Speedrunner vs Hunter mode for pressure-driven multiplayer sessions.
  • Herobrine event systems and hunter gameplay as priority content tracks.
  • Intelligent and brave companion AI bot built in for survival-focused players.

Updated command deck

Installation

Current baseline and setup

Use the latest stable 26.1 line dependencies and confirm compatibility on the release page.

1

Install Fabric or Quilt for Minecraft 26.1

Use the latest stable loader release for the 26.1 line.

2

Install required libraries

Install Fabric API or Quilted Fabric API, Fabric Language Kotlin, and GeckoLib.

3

Drop Aeonis into mods folder

Download the latest release jar and place it in your mods directory.

4

Boot and validate

Use /aeonis help or /aeonis sys ping in-game to verify load and system health.

Minecraft: 26.1 baseline 26.1.1: hotfix-compatible Java: 21+ runtime Fabric API, Kotlin, GeckoLib: required
