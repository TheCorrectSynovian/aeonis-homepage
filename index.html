Release Highlights

What changed in 4.0.0

Information below is remastered from README and release notes so the website reflects the current stable release.

Stable release milestone

First non-snapshot stable build on Minecraft 26.1, with final migration completed from 1.21.11.

Snapshot path completed

The 4.0 line stabilized from SNAPSHOT5 through SNAPSHOT10 before final 4.0.0 release cut.

Warning policy remaster

Global startup warning spam removed; warnings are now targeted to specific in-progress surfaces.

Prop Hunt hotfix remaster

Improved round-state validation, leave cleanup, border enforcement, pacing, and reward handling.

Copper Stalker remaster

Food-based temporary taming, morning departure behavior, and feeding-window pacification flow.

Fun command remaster

Added timed morph pranks and new commands: boing, confetti, and swap.

Deeper Dark dimension

Full-scale underworld expansion with custom cave logic and progression routing.

  • 7 biomes: Amethyst Mines, Ancient Dark, Deep Oasis, Deeper Dark, Deeper Dark Cavern, Obsidian Barrier, Volcanic Caverns.
  • Worldgen structures: Ancient Fortress, Ancient Village, Laboratory, Amethyst Mineshaft, Sculk Trap, and more.
  • Custom noise, placed features, fossil fields, lava systems, and sculk growth paths.

Ancard expansion and ecosystems

Ancard receives atmosphere, renderer, and event pipeline upgrades plus expanded creature rosters.

  • Custom fog and post-processing with dynamic lightning and eclipse events.
  • New block families and biome materials in bloodroot and veilshade regions.
  • Mobs include Ancard Sovereign, Ancient Colossus, Rift Screecher, Veil Mimic, and Arda-side sculk variants.

Installation

Current baseline and setup

Use the latest stable 26.1 line dependencies and confirm compatibility on the release page.

1

Install Fabric or Quilt for Minecraft 26.1

Use the latest stable loader release for the 26.1 line.

2

Install required libraries

Install Fabric API or Quilted Fabric API, Fabric Language Kotlin, and GeckoLib.

3

Drop Aeonis into mods folder

Download the latest release jar and place it in your mods directory.

4

Boot and validate

Use /aeonis help or /aeonis sys ping in-game to verify load and system health.

Minecraft: 26.1 baseline 26.1.1: hotfix-compatible Java: 21+ runtime Fabric API, Kotlin, GeckoLib: required
